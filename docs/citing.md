# Citing PertCF

If you use PertCF in your research, please cite the original paper:

## Paper citation (BibTeX)

```bibtex
@inproceedings{bayrak2023pertcf,
  title     = {PertCF: A Perturbation-Based Counterfactual Generation Approach},
  author    = {Bayrak, Bet{\"u}l and Bach, Kerstin},
  booktitle = {Artificial Intelligence XXXVIII},
  series    = {Lecture Notes in Computer Science},
  volume    = {14381},
  pages     = {174--187},
  year      = {2023},
  publisher = {Springer, Cham},
  doi       = {10.1007/978-3-031-47994-6_13}
}
```

## Software citation (BibTeX)

```bibtex
@software{bayrak2024pertcf_pkg,
  title   = {pertcf: A Python Package for Perturbation-Based Counterfactual Explanations},
  author  = {Bayrak, Bet{\"u}l},
  year    = {2024},
  url     = {https://github.com/b-bayrak/PertCF-Explainer},
  version = {1.0.0}
}
```

## APA format

Bayrak, B., & Bach, K. (2023). PertCF: A perturbation-based counterfactual generation approach. In *Artificial Intelligence XXXVIII*, LNCS 14381 (pp. 174–187). Springer. https://doi.org/10.1007/978-3-031-47994-6_13
